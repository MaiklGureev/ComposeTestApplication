# Гитхаб клиент

Функционал:

1. Сплэш срин
2. Главный экран
3. Поиск репозиториев
4. Поиск юзеров
5. Просмотр репозиториев
6. Просмотр веток
7. Скачивание и распаковка архивов
8. Открытие ссылок на гит
9. Загрузки

Стек:

- Kotlin
- Compose
- Dagger2
- Okhttp
- Retrofit
- Material Theme
- android 6-13
